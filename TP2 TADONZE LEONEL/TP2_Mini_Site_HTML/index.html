Introduction

Ce mini-site a été créé dans le cardre du cours de TP Programmation web

Il permet de découvrir la création de plusieurs pages HTML reliées entre elles

Objectif du mini-site

L'objectif est d'apprendre à structurer un petit projet web avec plusieurs fichiers.

Pages disponibles

  • La page d'accueil présente le contexte general de mon mini-site
  • La page de présentation qui contient les infos presonnelles de l'étudiant
  • La page contact qui contient les informations de contact de l'étudiant

En réalité, j'apprends la programmation web pour accéder à des métiers en forte demande et multiplier mes opportunités professionnelles.
