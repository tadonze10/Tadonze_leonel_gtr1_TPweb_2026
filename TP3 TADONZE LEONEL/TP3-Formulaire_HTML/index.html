Introduction

Ce mini-site a été crée dans le cadre du cours de Tp Programmation Web

Il permet de decouvrir la création de plusieurs pages HTML reliées entre elles.

Objectif dumini-site

L'objectif est d'apprendre a structurer un petit projet web avec plusieurs fichiers.

Pages disponibles

  • La page Accueil presene le contexte général du mini-site.
  • La page Présentation contient les informations personnelles de l'étudiant.
  • La page Contact contient les informations de contact de l'etudiant.
